# Moscow time server

This service shows current time in Moscow zone.

## Setup

```bash
#Create python venv
python3 -m venv venv

#Activate venv
source venv/bin/activate

#Install all required packages
pip3 install -r requirements.txt
```

## Run

```bash
#Activate venv
source venv/bin/activate

#Start server
python3 main.py
```
